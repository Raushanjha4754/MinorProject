## ğŸ“ Screenshot Specifications

- **Format**: PNG or JPG
- **Size**: 
  - Desktop: 1920x1080 or 1440x900 recommended
  - Mobile: Capture actual mobile viewport
- **Quality**: High resolution (at least 1920px width)
- **File Size**: Optimize images (< 500KB each if possible)

## ğŸ¨ Screenshot Guidelines

1. **Use Consistent Browser**: Take all screenshots in the same browser
2. **Clean State**: Use fresh data or seed data for consistency
3. **Hide Personal Info**: Blur or remove any personal/sensitive data
4. **Full Page**: Capture full page, not just viewport
5. **Light Mode**: Use light mode (matches NITJ theme)
6. **Browser Theme**: Use default browser theme for consistency
